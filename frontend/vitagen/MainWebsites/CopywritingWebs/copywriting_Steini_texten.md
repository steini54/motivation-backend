# Steini_texten.html — Copywriting & Redesign Plan

## Project Context

Page yang akan diredesign: `Steini_texten.html`

Goal halaman ini adalah menjadikan halaman **Texten / Product Portfolio** sebagai sales page yang jelas, profesional, dan konsisten dengan design system baru Syntext/VitaGen yang sudah dipakai di `index.html`.

Halaman lama sudah memiliki empat layanan utama:

1. Page-Upgrade
2. Bewerbungs-Kit
3. Werbe- & Marketingtexte / Impact Copy
4. Webseiten und SEO / On-Page Optimierung

Masalah utama halaman lama bukan idenya, tetapi struktur dan copywriting-nya masih terlalu panjang, kurang terarah, dan belum cukup kuat untuk conversion.

---

## 1. Analysis of Existing Page

### Existing Structure

Current `Steini_texten.html` berisi:

- Header navigation lama
- Main title: `Texte verfassen`
- Accordion/checklist style untuk 4 layanan:
  - 🛠️ Page-Upgrade
  - 🧩 Bewerbungs-Kit
  - 📣 Werbe- & Marketingtexte
  - 🎯 Webseiten und SEO
- Tiap layanan punya deskripsi panjang, list layanan, isi paket, faktor harga
- CTA ke `Steini_Kontakt.html`
- VitaGen CTA
- Footer legal links

### Existing Copywriting Problems

#### 1. Value proposition belum langsung jelas

Judul `Texte verfassen` terlalu umum. User belum langsung tahu:

- Syntext membantu siapa?
- Masalah apa yang diselesaikan?
- Layanan mana yang cocok untuk mereka?
- Apa next step yang harus dilakukan?

#### 2. Banyak teks bagus, tapi terlalu tersembunyi

Copy lama punya banyak ide bagus seperti:

- tone yang autentik
- grammar dan spelling yang bersih
- target audience
- text that creates attention
- SEO/on-page optimization

Tapi karena dikemas dalam accordion panjang, value utama tidak langsung terbaca.

#### 3. Beberapa copy terasa kurang premium

Beberapa phrase seperti:

- `WAs tRiggEerT?`
- `Marketing-Harfe`
- `Schlüsselwörter!!`
- `Fallen wir gemeinsam auf!`

punya karakter, tetapi kurang cocok dengan arah baru Syntext yang lebih profesional, Swiss/German, dan trust-oriented.

#### 4. CTA perlu lebih konsisten

CTA lama:

- `Page-Upgrade jetzt anfragen`
- `Bewerbungs-Kit jetzt anfragen`
- `Impact Copy jetzt anfragen`
- `On-Page Optimierung jetzt anfragen`

Sudah cukup jelas, tetapi secara conversion bisa diperkuat dengan CTA yang lebih calm dan advisory:

- `Projekt anfragen`
- `Bewerbungsunterlagen prüfen lassen`
- `Text optimieren lassen`
- `Website-Texte prüfen lassen`

#### 5. Harga belum perlu dibuat detail

Halaman lama menyebut bahwa harga tergantung:

- panjang teks
- jumlah halaman
- kompleksitas tema
- jumlah teks
- kondisi awal teks

Ini bagus untuk transparansi, tetapi sebaiknya dikemas sebagai **pricing note**, bukan berulang di setiap section.

## 2. Recommended Page Positioning

### DE

**Steini_texten.html should become the product portfolio page for Syntext writing services.**

Positioning:

> Professionelle Texte, die klarer wirken, besser verstanden werden und zur richtigen Handlung führen.

Syntext tidak hanya “menulis teks”, tetapi membantu user membuat komunikasi yang:

- verständlich
- glaubwürdig
- zielgerichtet
- stilsicher
- handlungsorientiert

### EN

**Steini_texten.html should become the product portfolio page for Syntext writing services.**

Positioning:

> Professional writing that reads clearly, builds trust and guides the reader toward the right action.

Syntext is not only “writing text”, but helping users create communication that is:

- clear
- credible
- audience-focused
- polished
- action-oriented

# 6. Design Direction for Implementation

The redesign should visually follow the already approved `index.html` direction:

- Same typography: Inter / system sans
- Same background style: soft neutral background with subtle radial accent
- Same color system: dark primary, muted text, gold accent
- Same topbar/navigation behavior
- Same footer style
- Same card language: white/transparent cards, thin borders, soft shadows
- Same CTA style
- Same mobile-first responsive behavior
- Same DE/EN data attribute system if implemented across redesigned pages

## Recommended Visual Sections

### Hero Visual Idea

Use CSS-based editorial mockup, similar to homepage visual language:

- floating document card
- highlighted corrected text line
- small grammar/check badge
- small SEO/tone badge
- no heavy image requirement

### Service Cards

Use 4 cards or 4 horizontal service blocks:

1. Page Upgrade
2. Application Kit
3. Advertising & Marketing Copy
4. Websites & SEO

Each card should include:

- title
- short value statement
- 3–5 bullet highlights
- CTA
- optional expanded detail section

### Process Section

Dark section like existing redesigned pages:

- 4 steps
- Anfrage → Bedarf klären → Text vorbereiten → Finalisieren

### VitaGen Bridge

Keep VitaGen as a supportive product CTA, especially under Bewerbungs-Kit, but do not make the whole page feel like VitaGen.
