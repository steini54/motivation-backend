# Syntext Main Website Copywriting Plan

Scope: `frontend/vitagen/MainWebsites/index.html` and `frontend/vitagen/MainWebsites/Steini_Dienstleistung.html`

Goal: full UI/UX refactor with new professional Syntext/VitaGen direction, while keeping the current page structure, key routes, analytics, legal/footer links, and service intent.

## 1. Positioning

Syntext should feel like the professional service layer around VitaGen: precise language, clear structure, practical AI support, and Swiss/German reliability. VitaGen can feel more product-led and builder-like; Syntext should feel more advisory, editorial, and service-led.

Core promise:

- DE: Klare Texte, saubere Struktur und professionelle Kommunikation - für Bewerbungen, Unternehmen und digitale Auftritte.
- EN: Clear writing, structured messaging and professional communication - for applications, businesses and digital presence.

Tone:

- DE: Swiss/German professional, direct, warm, no slang, no overpromising.
- EN: calm, clear service language, not literal word-for-word translation.
- Avoid: "AI magic", "super fast", "instant", aggressive sales CTAs.
- Prefer: "KI-Unterstützung", "professionell vorbereiten", "Vorschau prüfen", "klar formulieren", "secure payment".

## 4. Bilingual Implementation Notes

- Default language should remain DE unless the current site already stores another preference.
- Add an explicit DE/EN switch in the header if implementation scope allows it.
- Use semantic copy keys instead of duplicating large hidden blocks where possible.
- Do not translate legal labels loosely: Impressum, Disclaimer, AGB should remain legally recognisable. English labels can be displayed as helper text only if needed.
- Preserve current primary nav routes: Dienstleistungen, Konditionen, Kontakt, About, Shop, DiMe.

## 5. Image Direction for Later Imagegen Step

Use custom generated images, not generic stock.

Index hero image brief:

- Professional editorial desk scene with layered paper, clean website wireframe fragments, subtle Swiss-inspired grid, neutral light background, deep green accent, no visible brand text, no fake UI text.

Dienstleistungen hero image brief:
